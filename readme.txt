== Description ==

Easily change the "Howdy" verbiage in the top right corner of your admin screen to anything else you'd like for all users.

== Installation ==

The install is incredibly easy

1. Upload the plugin files to the `/wp-content/plugins/rs-change-howdy-verbiage` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the new "Raineri Software" tab on the left (all of our plugins will be centralized here for ease of use)
4. Click the "Change Howdy" sub menu
5. Input whatever text you'd like to change it to and press "Save Settings"

And you're all set!


== Frequently Asked Questions ==

Allows you to update the "Howdy" verbiage in the top right of the admin screen.

= What's the easiest way to install it? =
